# Customer Churn Analysis - Python

## Objective
Identify key factors causing customer churn and build
a prediction model using machine learning.

## Tools
- Python (Pandas, Matplotlib, Seaborn)
- Scikit-learn (Logistic Regression)
- Jupyter Notebook

## Dataset
Telco Customer Churn Dataset (Kaggle)
7,000+ customer records, 20 features

## Key Analyses
- Churn rate by contract type, tenure, charges
- Correlation analysis of churn factors
- Logistic regression prediction model
- Model accuracy and confusion matrix
